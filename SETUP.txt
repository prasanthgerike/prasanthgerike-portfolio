‚úèÔ∏è CUSTOMIZE YOUR PORTFOLIO:
================================================================================

1. CHANGE YOUR NAME:
   - Open index.html with any text editor
   - Find: <h1 class="hero-title">Prasanth <span class="gradient-text">Gerike</span></h1>
   - Replace with your name

2. UPDATE COLORS:
   - Open styles.css
   - Find: :root { (around line 8)
   - Change color values:
     --primary: #0ea5e9;      (Main blue)
     --accent: #8b5cf6;       (Purple accent)
     --background: #0f172a;   (Dark background)

3. EDIT CONTENT:
   - Open index.html
   - Find sections: Experience, Projects, Skills
   - Edit text directly

4. CHANGE FONTS:
   - Open index.html
   - Find: <link href="https://fonts.googleapis.com...
   - Replace with your font URL

5. ADD IMAGES:
   - Create an "images" folder in this directory
   - Add your images
   - Update image paths in CSS
